Grok API EquityQuery Theme Screening New Feature Proposal

Theme-Based Screening
× Grok-Driven Theme Discovery

A 2-stage pipeline that asks Grok "which sectors are hot right now?" then screens for undervalued stocks within that theme

Evolving screening from "condition input" to "intent input."
When the user says "find undervalued stocks in today's hottest themes," Grok detects trending themes and EquityQuery narrows down quality stocks within those themes. Investment candidates are delivered automatically by reading the market's pulse.

🔄
2-Stage Pipeline
Step 1
Detect Trends with Grok
Retrieve today's notable sectors and themes in natural language by scanning X and the Web
Grok API
Step 2
Map Theme Names
Convert Grok's response to theme keys in themes.yaml (ai / ev / defense, etc.)
themes.yaml
Step 3
Screen by Theme × Preset
Convert theme-corresponding industries to EquityQuery and AND-combine with existing preset criteria
EquityQuery
Step 4
Output with Theme Context
Present the reason "why this theme now" together with stock scores
Report
